# 7. Install the core apps

## 7.1 Install the core apps

*   Claude Desktop ([see](/Jelly-Stack/Claude/Getting-started-guide))
*   OpenCode Desktop ([see](/Jelly-Stack/OpenCode/Getting-started-guide))
*   VS Code + OpenCode extension
*   GitHub CLI (optional but useful)
*   Azure CLI

## 7.2 Install the Azure CLI on Windows

```
winget install --exact --id Microsoft.AzureCLI
```

```
az --h
```

## 7.3 Install GitHub CLI

```
https://cli.github.com/
```

```
gh --h
```
